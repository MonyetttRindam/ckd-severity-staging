# Raw Data

The dataset file (`CKD_NHANES_2021_2023.csv`) is not included in this repository due to size and licensing considerations.

## How to obtain the dataset

1. Download from Kaggle: [CKD NHANES 2021-2023 Staged Kidney Disease](https://www.kaggle.com/datasets/alitaqishah/ckd-nhanes-2021-2023-staged-kidney-disease)
2. Extract the CSV file
3. Place `CKD_NHANES_2021_2023.csv` in this directory (`data/raw/`)

## Source

- **Curated by:** Ali Taqi Shah ([Kaggle profile](https://www.kaggle.com/alitaqishah))
- **Original source:** U.S. National Health and Nutrition Examination Survey (NHANES) 2021-2023 cycle
- **Staging methodology:** Applied based on eGFR (estimated glomerular filtration rate) and albuminuria criteria
